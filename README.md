# Interclasse Digital

Aplicativo mobile desenvolvido em React Native para gerenciamento de jogos do interclasse.

Funcionalidades
- Modalidades esportivas
- Tabela de classificação
- Cadastro de salas
- Calendário de jogos
- Armazenamento local com AsyncStorage
- Integração com API usando Axios

Tecnologias
React Native
Expo
Axios
AsyncStorage
